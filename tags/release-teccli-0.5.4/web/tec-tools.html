MODNLP: tec-tools: tec-client/tec-serverv2

TEC (short for "Translational English Corpus") is a computerised collection of contemporary translational English text collected and maintained at The University of Manchester and made available on the Internet by Trinity College Dublin.

The TEC processing software (tec-client and tec-server) were originally developed to allow free access to the corpus over the Internet. These tools are currently being revamped and integrated into modnlp Preliminary versions of the server and client software can be downloaded as alpha releases.

In order to access the server or local corpora you will need to download the modnlp-teccli module.

In order to create an index that you can access through the TEC browser (modnlp-teccli) you will need modnlp-idx (the indexer).

If you want to make corpus data and concordances (though not necessarily your full texts) available to the community over the Internet your should download the modnlp-tecser module.

If you would like to test and/or use the software, a binary distribution of the entire suit is available for downloading. Simply uncompress the archive and follow the instructions in the README file.

The old distributions of TEC are available (under the GPL) at the TEC software web site, where you can also try the concordancer (via Java Web Start).

More information on the TEC client/server architecture can be found in the following paper:
